Non-Immigrant M (Media) — для аккредитованных журналистов, корреспондентов, съёмочных групп с заданием в Таиланде. Одобрение PRD/DOI до визы. Разрешает медиа-работу. Не для блогеров без пресс-аккредитации.

Что такое Non-M

Тип M ведёт PRD (Department of Public Relations) — не для обычных блогеров и туристов с камерой.

  • Иностранные СМИ с бюро в Таиланде
  • Документальные проекты с контрактом
  • Пресс-карта / аккредитация дома

Endorsement PRD

  1. Тайский спонсор: СМИ, продакшн
  2. Заявка в PRD, Бангкок
  3. После одобрения — Non-M в консульстве
  4. Work permit часто для медиа-деятельности
  5. Срок = длительность проекта

Non-M vs Non-B vs DTV

Non-B: тайский работодатель — Non-B.
DTV: удалёнка — DTV.
Non-M: только пресса/док с письмом PRD.

Паттайя

Мало кто живёт в Паттайе на Non-M — чаще Бангкок, Паттайя для сюжетов. Короткие визиты + re-entry для съёмочных групп.

Документы

  • Письмо PRD
  • Контракт / assignment letter
  • Пресс-удостоверение
  • Паспорт, фото, документы спонсора

Отказы

  1. YouTube без аккредитации
  2. Нет PRD
  3. Не медиа-проект

Российские журналисты с аккредитацией: тот же маршрут PRD → консульство — не путать с туристической съёмкой без разрешений.

TM30 и 90-day report обязательны даже для коротких проектов в Паттайе.

Срок визы привязан к контракту — после окончания съёмок нужен новый статус (турист, DTV) или выезд.

Фото/видео для коммерции без PRD — риск штрафа; для прессы только Non-M с письмом.

Сроки

PRD 2–4 недели, консульство 1–2 недели. Съёмки в Паттайе планируйте с запасом.

Блогеры

Инфлюенсеры без прессы: DTV или турист. «Media-виза для YouTube» от агентов — часто скам.

Аккредитация

Пресс-карта, письмо редакции, план сюжета — PRD проверяет реальную журналистику. Фриланс без редакции — почти невозможно. Корреспонденты с бюро в Бангкоке продлевают визу у PRD на Charoen Krung; жители Паттайи сдают 90-day report в Джомтьене, но endorsement всё равно через Бангкок.

Съёмки в EEC/Паттайе: короткий stay + re-entry, TM30 от отеля.

Бюджет: консульство + PRD + permit — планируйте до приезда группы.

Продление

Новое письмо PRD для продления. Конец проекта — смена статуса. Overstay — штраф. PRD в Бангкоке — будни.

Документалисты: уточняйте, что сюжет — медиа, не коммерческая реклама без лицензии.

Кино без прессы — Non-B через тайскую компанию. Агенты «медиа-виза за 3 дня» в Паттайе — красный флаг. TDAC при въезде обязателен.

FAQ

Инфлюенсер?

Нет — DTV или турист.

Срок?

90 дней — 1 год по проекту.

Семья?

Dependent отдельно.

Work permit?

Часто для медиа-работы в пакете.
